# 🌿 Plantasia 

Un gestionnaire de catalogue de plantes développé en C++ pour suivre et entretenir facilement les plantes de votre balcon.

## 📖 Description

Plantasia est une application simple et légère qui permet de recenser toutes les plantes présentes sur un balcon. Elle aide à garder une trace des besoins spécifiques de chaque plante (arrosage, exposition au soleil, type d'engrais) pour s'assurer qu'elles s'épanouissent dans les meilleures conditions.

## ✨ Fonctionnalités (Prévues ou Actuelles)

* **Ajouter une plante** : Enregistrer une nouvelle plante avec son nom, son espèce et ses besoins.
* **Lister les plantes** : Afficher un inventaire complet des plantes du balcon.
* **Suivi de l'arrosage** : Savoir quand une plante a été arrosée pour la dernière fois et quand est le prochain arrosage.
* **Supprimer/Modifier** : Mettre à jour les informations d'une plante ou la retirer du catalogue.
* **Sauvegarde des données** : Sauvegarde locale dans un fichier (JSON, CSV ou TXT) pour ne pas perdre les données entre chaque utilisation.

## 🛠️ Prérequis

Pour compiler et exécuter ce projet, vous aurez besoin de :

* Un compilateur C++ compatible avec C++17 (GCC, Clang ou MSVC).
* [CMake](https://cmake.org/) (version 3.10 ou supérieure) pour la gestion de la compilation.
